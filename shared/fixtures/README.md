# Fixtures Directory

This directory contains live fixture data updated throughout the day.

## Files

- `upcoming.json` - Today's and tomorrow's fixtures
- `live.json` - Currently live matches
- `metadata.json` - Update timestamp and fixture counts

## Update Schedule

Fixtures are updated **every 2 hours** by the GitHub Actions workflow.
